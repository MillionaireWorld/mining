# Mining
mining for millionaire world game.
# Install
git clone https://github.com/MillionaireWorld/mining.git
cd mining
npm install
# Quick Config
* 1.vim index.js
* 2.replace defaultPrivateKey to your eos account privatekey.
* 3.replace fromaccount to your eos account.
* 4.sh ./run.sh
